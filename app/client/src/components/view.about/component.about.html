Welcome to Foobarbot Developer Preview 1.1

Hows and whys about the app

Teach by learning

Foobarbot is a code snippet manager and code search engine created for developers. Users can create code snippets for later usage, share them with each other, or search for problems and find solutions across snippets uploaded by other users or find search results pulled from GitHub and Stack Overflow.

The appplication is currently in developer preview state and is being tested by its developers.

Developer Preview 1.1

  • Snippets fetched from GitHub and starred by any user gets posted by Foobarbot
  • Snackbar component improved
  • Snippet and snippet list components introduced
  • Pagenation: show more button on snippet lists
  • TypeScript issues eliminated
  • Minor bugfixes
  • Cosmetic updates

Developer Preview 1.0

  • Users can register an account and log in.
  • Users can create new snippets
  • Users can star other snippets and display them on their profile
  • Users can search and browse between snippets uploaded by others
  • Users can search and browse between snippets fetched from GitHub Gists
  • Users can update their profiles or delete their accounts.
  • Users can simulate notifications

Product roadmap:

  • Organize your snippets into collections and create tutorials
  • Refine search engine
  • Implement Stack Overflow API
  • Implement YouTube API
  • Save snippets instantly with the Chrome extension
  • Insert snippets instantly with Atom/Sublime packages
