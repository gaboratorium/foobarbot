Welcome to Foobarbot Developer Preview

Hows and whys about the app

One web app to rule them all

Foobarbot is a single page application written in NodeJs and VueJs. The application is currently in developer preview state and is being tested by its developers.

Product roadmap:

  • Save and organize your code snippets, collections and tutorials
  • Browse between templates and tutorials created by other developers
  • Read tutorials created by the community
  • Save snippets instantly with the Chrome extension
  • Insert snippets instantly with Atom/Sublime packages
